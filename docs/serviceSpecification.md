## Request Headers

The following headers are mandatory:

| Header          | Value                  |
|----------------|------------------------|
| `Accept`       | application/json       |
| `Content-Type` | application/json       |
| `Authorization` | OAM-Auth <access-token> |

See the *Authentication and Authorization* section for details.
